1. 初始化项目
   npm init -y
   npm install --save-dev typescript @types/node

2. 配置 TypeScript 创建 tsconfig.json 文件：
   npx tsc --init
- 重要配置
   {
   "compilerOptions": {
   "target": "ES6", // 编译为 ES6 兼容代码
   "module": "commonjs", // 使用 CommonJS 模块规范（Node.js 默认）
   "outDir": "./dist", // 输出目录
   "rootDir": "./src", // 源代码目录
   }
   }

4. 编写 TypeScript 代码
示例：src/index.ts

5. 优化开发体验
配置 package.json 脚本

{
  "scripts": {
    "build": "tsc",
    "start": "node dist/index.js",
    "dev": "ts-node src/index.ts",
    "watch": "tsc --watch"
  }
}
